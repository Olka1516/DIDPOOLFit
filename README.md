# DIDPOOL Fit App

DIDPOOL Fit – додаток для тренувань, створений на Flutter. Додаток буде дозволяти користувачам реєструватися, логінитися, переглядати свій профіль, шукати, відслідковувати та додавати результати вправ

Посилання на дизайн Figma: https://www.figma.com/design/0Ceobphtx1ZysGrFpK5l6B/Untitled?node-id=15-1147&p=f&t=P7jvfgT2h3eTq5Aw-0

# Сторінки

- Welcome Page – стартова сторінка
- Onboarding Page – сторінка з невеликим знайомством із додатком
- Register Page – форма для створення нового акаунту користувача
- Login Page – форма для входу в акаунт
- Workout Page – створення результати вправ користувача
- Edit Page – редагування персональної інформації користувача
- Home Page – головна сторінка користувача з пошуком та результатами вправ
- Profile Page – відображення профілю користувача

# Основні компоненти

- back_button - кастомна кнопка назад
- custom_action_button – кастомна кнопка з градієнтом
- custom_app_bar – кастомна верхня навігаційна панель з підсвічуванням активного елементу
- custom_list – список вправ з іконками та даними
- info_card – відображення персональної інформації користувача
- text_field – текстові поля з іконками або без
